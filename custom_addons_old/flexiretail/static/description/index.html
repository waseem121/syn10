Installation

  • Download fonts : http://www.reportlab.com/ftp/fonts/pfbfer.zip
  • now put this fonts into location : /usr/lib/python2.7/dist-packages/reportlab/fonts/(put the files) note : if fonts folder does not exists, create that folder and copy that fonts into it
  • To see the preview of label install following packages
  • sudo apt-get install python-wand
  • sudo apt-get install libmagickwand-dev
  • sudo apt-get install imagemagick

Delete POS Orders.

Module allows to delete pos orders which are in paid/posted state.


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Delete single/multiple pos orders which are in paid/posted state.
  • After delete pos orders it will affect in the statements of pos session.
  • After delete pos orders it will remove the accounting entries as well.
  • After delete pos orders it will remove delivery order as well.

How to use

  • Configure security pin and give allow user for delete paid/posted pos orders.


  • User can select multiple orders and delete from list view.


  • User can delete single order from for view of pos order.


  • After select pos orders user will get pop up where they can enter pin and process of delete an orders.

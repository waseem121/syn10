Odoo 10 Point of Sale Extension

Allow user to check product quantity in different warehouse from Point of Sale.


Note: This module is tested and working in Odoo community edition & Ubuntu OS.

How to use

  • Here we can see the total quantity of products as default odoo feature.


  • Now in Point of Sale, we have a button `Product Quantity'. This button will display quantity in different warehouse of selected product from the list.
